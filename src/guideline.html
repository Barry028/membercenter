設計規範

此為 設計規範，SCSS 檔案主要分成 5 個資料夾： (最後會再進行合併)

  • Helpers

    設定網站設計時用到的全域變數，字型等等，例如顏色、斷點、字型等等變數設定

  • Base

    Css Reset 以及一些基本設定

  • Layout

    格線系統以及頁面的共用區塊

    • 會員中心
  • Components

    元件類的樣式都放在這邊

  • Pages

    頁面的樣式都在這邊
